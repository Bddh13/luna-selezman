# CONTEXT.md — LÚNA SELEZMAN

## Project
One-page portfolio site for visual artist LÚNA SELEZMAN.

## Sections
1. Hero — fullscreen black, name "LÚNA SELEZMAN" large uppercase, 
   short tagline, CSS glitch animation on load
2. Gallery — asymmetric masonry grid, 10 placeholder works with titles
3. Contact — email + Instagram link + "Commission a work" button

## Visual Direction
Dark editorial with glitch undertones.
References: albertomielgo.com, ghostline.live, orkhan.art

Rules:
- Background: #080808
- Text: #e8e4dc
- Accent: #c8f542 (electric lime)
- Font headings: Space Grotesk, uppercase
- Font name: Cormorant Garamond, large
- NO rounded corners (border-radius: 0)
- NO shadows, NO gradients
- Thin 1px #222 borders as dividers
- Images: object-fit cover, full container width

## Micro-interactions
- Hero: CSS glitch effect on name, plays on load then stops
- Gallery: fade-in stagger on scroll
- Hover on gallery item: dark overlay + title appears

## Tech Stack
Vite + React + Tailwind CSS
Google Fonts: Space Grotesk + Cormorant Garamond
